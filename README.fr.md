## 🚀 Mes Projets

Voici quelques projets sur lesquels j'ai travaillé :

* 🚢 **[CAN - Boat Booking System](https://github.com/AgentOutsiders/maritime-booking-system)** : Projet scolaire de première année réalisé en binôme. Ce système de réservation comprend une application console en **C#** et un site web en **HTML/CSS/JS** pour générer les factures et les cartes d'embarquement avec QR Code.
* 🎯 **[MIPS32 Battleship Simulation](https://github.com/AgentOutsiders/mips32-battleship-simulation)** : Une simulation automatisée de Bataille Navale programmée entièrement en **Assembleur MIPS32**. Ce projet génère une grille dynamiquement et utilise un algorithme de ciblage récursif pour couler les navires.
* 🧹 **[Rustidy](https://github.com/AgentOutsiders/Rustidy)** : Un outil de ligne de commande (CLI) rapide codé en **Rust**. Il permet d'organiser automatiquement les fichiers d'un dossier en les triant dans des sous-dossiers selon leur extension.
* 🏰 **[Tower-Defense](https://github.com/AgentOutsiders/tower-defense)** *(En cours de développement 🚧)* : Un projet de jeu de type Tower Defense, actuellement développé avec **Godot Engine** et **C#**.
